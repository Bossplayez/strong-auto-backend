GitHub Можливості

Модулі платформи

🚗

Каталог авто

Управління каталогом транспортних засобів з детальними характеристиками, фото та фільтрацією.

🧮

Калькулятор вартості

Розрахунок повної вартості авто з урахуванням доставки, розмитнення та додаткових витрат.

🔗

Copart інтеграція

Автоматичний імпорт даних з аукціону Copart — лоти, ціни, стан авто.

📋

Лід-менеджмент

Збір та обробка заявок від клієнтів, автоматичні сповіщення та трекінг статусів.

🔐

Авторизація

JWT-аутентифікація з ролями (Admin, Manager, User), рефреш-токени та гарди.

📢

Сповіщення

Email та push-нотифікації, розсилки по сегментах користувачів, новини.

API Endpoints

  • POST /auth/login Авторизація
  • GET /catalog/vehicles Список авто
  • POST /calculator/calculate Розрахунок вартості
  • GET /copart/lots Лоти Copart
  • POST /leads Створити заявку
  • GET /users/profile Профіль користувача
  • PUT /settings Налаштування
  • GET /news Новини
